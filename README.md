# Complaint Management System

## Project Description
A Java-based Complaint Management System developed using Core Java, JDBC, Oracle Database, and HTTP Server. The system automatically categorizes complaints and assigns priorities based on the complaint description.

## Technologies Used
- Core Java
- JDBC
- Oracle Database
- HTTP Server
- Gradle

## Features
- Add Complaint
- View Complaints
- Auto Category Detection
- Auto Priority Assignment
- Oracle Database Integration
